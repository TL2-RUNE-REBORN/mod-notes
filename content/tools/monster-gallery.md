## 进入图鉴

**[▶ 打开怪物图鉴](/tools/monsters/)** —— 无需安装,浏览器直接用。

## 是什么

原版《火炬之光2》全部怪物,加上挑战者大陆整合包各组件(暗黑传奇、群魔堕落、职业技能、佣兵系统……)
新增的怪物与召唤物,从游戏 `DAT` 源数据直接提取生成:

- **数值**:生命 / 伤害 / 护甲 / 元素护甲 / 经验,按引擎真实语义显示为
  **等级曲线倍率**(逆向确认:DAT 值 ÷100 × 等级曲线,BOSS 与 Champion 走 Champion 曲线);
- **实战换算**:详情页内置原版(105 级)与挑战者(180 级)两套等级曲线,
  拖动等级滑条、切换休闲 / 普通 / 老兵 / 精英难度,直接算出该等级下的实际生命、伤害与击杀经验;
- **技能**:怪物技能表含释放几率与 AI 触发技(低血召唤、暴走等),技能名链到游戏内中文名;
- **特性与词缀**:格挡 / 暴击 / 施法速度等自带特性,以及 BOSS / Champion 的控制抗性词缀
  (昏迷 / 减速 / 击退抗性……)按游戏内文案渲染;
- **3D 预览**:怪物模型逐子网格解析材质贴图在线查看,点击任意一行即可旋转把玩;
- **变体折叠**:同名随机变体(如群魔堕落一万个随机装备的"堕落英雄")自动折叠为一条,数值取区间。

## 怎么用

顶栏按来源(原版 / 各组件 MOD)、类别(BOSS / Champion / 普通怪 / 宠物召唤)、家族筛选,
或直接搜索怪物名;点击表格任意一行打开详情、3D 模型与实战数值计算器。
数据随整合包版本更新重新生成,以游戏内实际数值为准。
