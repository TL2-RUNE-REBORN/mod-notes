## Enter the gallery

**[▶ Open the Equipment Gallery](/tools/equipment/)** — nothing to install, runs in the browser (interface currently in Chinese).

## What it is

Equipment data from every component of the Challenger Continent mod pack (Diablo Legacy, Fallen Horde, Amulets, Supreme Compatibility, …), extracted straight from the mods' `DAT` source data:

- **Stats**: level, rarity, requirements (strength / dexterity / focus / vitality), damage / armor;
- **Socketables**: gems / skulls / eyes / embers / lucky coins, with effects listed per socket target (weapon / armor / trinket);
- **Effects & affixes**: innate item effects and fixed affixes, value ranges rendered exactly as the in-game tooltip; trigger effects link to the actual skill;
- **Sets**: member list (cross-linked) plus the bonus at each piece-count tier;
- **3D preview**: weapons use their drop models, armor uses the humanoid wardrobe mesh — click any row to view.

## How to use it

Filter by rarity / type / source mod in the top bar, or search by item name; click any table row for details and the 3D model. Data is regenerated with each mod-pack release — in-game values are authoritative.
